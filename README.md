# 🛒 Market Basket Analysis with Python

A Python-based data analytics project that explores customer shopping behavior using an e-commerce dataset. The project performs data cleaning, exploratory data analysis (EDA), customer segmentation, clustering, statistical analysis, and visualization to uncover valuable business insights that can improve customer experience and marketing strategies.

---

## 📌 Project Overview

Understanding customer purchasing behavior is essential for improving product recommendations, increasing customer satisfaction, and enhancing business performance.

This project analyzes customer demographics, purchase frequency, shopping satisfaction, recommendation systems, browsing behavior, and cart abandonment patterns using Python.

---

## 🚀 Features

* Cleaned and preprocessed raw e-commerce data
* Performed Exploratory Data Analysis (EDA)
* Analyzed customer demographics and purchase behavior
* Identified popular product categories
* Studied cart abandonment factors
* Evaluated customer satisfaction and recommendation effectiveness
* Built customer segments based on shopping behavior
* Applied K-Means clustering for customer segmentation
* Created insightful visualizations for business decision-making
